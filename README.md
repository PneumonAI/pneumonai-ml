# PneumonAI ML

Training, evaluation, explainability, and export for PneumonAI MVP v1.

## MVP responsibilities

- Prepare one public chest X-ray dataset.
- Train and evaluate one ResNet18 pneumonia classifier.
- Generate Grad-CAM heatmaps and overlays.
- Export the selected model as TorchScript.
- Verify PyTorch/libtorch inference parity.

Linear owns roadmap planning. GitHub issues in this repository own executable ML work. Datasets, private medical data, secrets, and large model artifacts must not be committed to Git.

See the [organization profile](https://github.com/PneumonAI) for project scope and resources.
